2. What is the difference between a method and function?
A function is a piece of code that can be called directly on the global scope (window) and run a defined algorithm
Ex: function myFunction(){
//statements;
}
To call it we can do this:
myFunction();
or
window.myFunction();
In the meanwhile a method is almost the same thing as a function with the difference that a method is defined inside of a user defined object and should be called on this object.
Ex:var myObjectPrototype= {
a:function myFunction(){
}
}
var myObject= Object.create(myObjectPrototype);
myObject.myFunction();
3. What does 'this' refer to when used in a Java method?
It means the actual object on which we are working.
4. What does 'this' refer to when used in a JavaScript method?
When called in JavaScript method, 'this' refers to any derived object on which the method is called.
5. What does 'this' refer to when used in a JavaScript constructor function?
When used in JavaScript constructor function, 'this' refers to every object that is going to be constructed from this constructor.
6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to?
"This" refers to the object on which the function is called in this case it refers to x.
7. What is a free variable in JavaScript?
It's a variable referred to by a function that is not one of its parameter or local variable.
