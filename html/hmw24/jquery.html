1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent.
//code fragment 1
$("li").each(function(idx, e) {
$(e).css(“color”, “yellow”); });
//code fragment 2
$("li").each(function() {
$(this).css(“color”, “yellow”); });
//code fragment 3
$("li").each(function(idx) {
$(this).css(“color”, “yellow”); });
The 3 code fragments are equivalent:
- In the first one, we pass the two parameters that the function needs, the first parameter is the index of the current element and the second parameter is the element itself. In this case any thing that we do on this second parameter will be done on the current element.
- In the second one, we don't pass any parameter to the function, that is not a problem and we apply the style on $(this). It's the same thing, because $(this) represents the current element.
- In the last one, we pass only one parameter, that is the index. No problem again, and we apply the style on this that represents the current element.
In conclusion, the 3 code fragments are equivalent.
